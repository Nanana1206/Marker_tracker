# 🎨 Учёт маркеров для раскрашивания

Веб-приложение для учёта маркеров (GuangNa, Languo, Grasp и др.) с поиском замен между брендами.

## Быстрый старт

1. Откройте файл `index.html` в браузере (двойной клик или перетащите в окно браузера)
2. При первом запуске инвентарь будет пуст — добавляйте маркеры через вкладку «➕ Добавить/Расход»
3. Доступны все бренды (GuangNa, Languo, Grasp) с полной таблицей цветов и соответствий

> **Сервер не нужен** — приложение работает полностью локально в браузере.

## Возможности

| Вкладка | Описание |
|---------|----------|
| 📦 **Мои маркеры** | Полный инвентарь с фильтрами, поиском, inline-редактированием количества |
| 🔍 **Поиск замен** | Найти аналог цвета из другого бренда (таблица соответствий из PDF-палитры) |
| ➕ **Добавить/Расход** | Быстрое добавление покупок или списание использованных маркеров |
| 📋 **История** | Журнал всех операций с маркерами |
| 📊 **Статистика** | Графики использования, топ цветов, рекомендации |
| 🛒 **Список покупок** | Автоматический список закончившихся маркеров с приоритетами |

## Дополнительные функции

- 🌙 Тёмная тема (кнопка в правом верхнем углу)
- 💾 Экспорт/импорт данных в JSON
- 📊 Экспорт инвентаря в CSV (для Excel)
- 🔄 Сброс к начальным данным
- ✏️ Добавление своих брендов и цветов
- 🔗 Ручное добавление соответствий между цветами

## Хранение данных

Все данные хранятся **локально в браузере** (localStorage). Никакая информация не отправляется на сервер.

- Чтобы **сохранить** данные: нажмите 💾 в панели инструментов → скачается JSON-файл
- Чтобы **загрузить** данные: нажмите ⬆️ → выберите ранее сохранённый JSON-файл
- Чтобы **сбросить** данные: нажмите 🔄 → все маркеры вернутся к начальному состоянию

## Структура файлов

```
index.html        — основная страница приложения
styles.css        — стили (светлая и тёмная тема)
app.js            — основная логика, управление UI
data.js           — начальные данные о маркерах
color-mapping.js  — таблица соответствий цветов между брендами
inventory.js      — управление инвентарём (добавление, редактирование)
features.js       — дополнительные функции (wishlist, графики, CSV)
```

## Совместимость

Работает в современных браузерах: Chrome, Firefox, Edge, Safari.
